# 💤 Dave's LazyVim Configuration

A starter template for [LazyVim](https://github.com/LazyVim/LazyVim).
Refer to the [documentation](https://lazyvim.github.io/installation) to get started.

Several things to run prior to this working:

```bash
brew install fzf ripgrep fd
brew install --cask font-jetbrains-mono-nerd-font
brew install --cask iterm2
```

You can use whatever nerdfont you want, but its required for the symbols.
Checkout [NerdFonts](https://www.nerdfonts.com/font-downloads)

You will need to change some settings in iterm2

First change the font:

- Settings > Profile > Text > Font > Select the nerdfont you downloaded.

Second change the following selection to prevent screen resize issues:

- Settings > Profile > Terminal > Emulation Features > Checkmark on Disable session-initiated window resizing
